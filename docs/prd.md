# PRD - F1 Teams Hub

## Identificação
* Nome: Kauan Felipe Kaveski Rutecki
* Projeto: F1 Teams Hub

## Descrição
Aplicação web para visualizar informações sobre equipes de Fórmula 1, como nome, país, títulos e pilotos.
Também exibe a classificação atual do campeonato utilizando uma API.

## Atores
* Visitante

## Histórias de Usuário

* Como visitante, quero ver a lista de equipes.
* Como visitante, quero ver os detalhes de uma equipe.
* Como visitante, quero ver os pilotos de cada equipe.
* Como visitante, quero ver a classificação atual da Fórmula 1.
* Como visitante, quero navegar entre as páginas do site para acessar diferentes informações.
* Como visitante, quero visualizar imagens das equipes para ter uma melhor experiência visual.
* Como visitante, quero identificar facilmente o nome e o país de cada equipe.
* Como visitante, quero ver os pilotos organizados por equipe para entender melhor a formação atual.
* Como visitante, quero acessar rapidamente a classificação do campeonato através do menu.
* Como visitante, quero visualizar os dados de forma organizada para facilitar a leitura.
* Como visitante, quero acessar o site pelo celular ou computador para poder usar em diferentes dispositivos.
* Como visitante, quero ver atualizações do campeonato para acompanhar a temporada.

## Funcionalidades

* Listagem de equipes
* Página de detalhes
* Exibição de pilotos
* Classificação via API
