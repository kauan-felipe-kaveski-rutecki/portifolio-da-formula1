# SDD - F1 Teams Hub

## 🎯 Visão Geral
O F1 Teams Hub é uma aplicação web voltada para a visualização de informações sobre equipes de Fórmula 1, incluindo dados como nome, país, pilotos e classificação atual.

A interface foi projetada com foco em uma experiência moderna, responsiva e visualmente alinhada com a identidade da Fórmula 1.

---

## 🎨 Design System

### Cores
- Primária: #E10600 (vermelho Fórmula 1)
- Secundária: #FFFFFF (branco)
- Terciária: #0163FF (azul de destaque)
- Neutra: #0B0B0B (preto escuro)

### Tipografia
- Títulos: Space Grotesk
- Texto: Inter

### Estilo Visual
- Interface com tema escuro (dark mode)
- Alto contraste para melhor legibilidade
- Destaques em vermelho para ações e elementos importantes
- Layout inspirado em dashboards modernos

---

## 🧩 Framework CSS
- Bootstrap 5

O framework será utilizado para:
- Sistema de grid responsivo
- Componentes reutilizáveis
- Padronização visual da interface

---

## 🌐 API Pública
- Ergast API

A API será utilizada para obter:
- Informações das equipes
- Dados dos pilotos
- Classificação atual do campeonato

---

## 🖥️ Estrutura da Interface

### Páginas principais
- Dashboard (visão geral)
- Equipes (grid de equipes)
- Classificação

### Layout
- Navbar superior (identidade e navegação principal)
- Sidebar lateral (menu de navegação)
- Área principal com conteúdo dinâmico

---

## 📱 Responsividade
A aplicação será totalmente responsiva, adaptando-se para:
- Dispositivos mobile
- Tablets
- Desktop

Será utilizado o sistema de grid do Bootstrap para garantir a adaptação dos layouts.

---

## 🧱 Componentes do Sistema

Os seguintes componentes foram definidos no protótipo e serão implementados com o framework:

- Navbar (barra superior)
- Sidebar (menu lateral)
- Cards (exibição das equipes)
- Botões (Primary, Secondary, Outlined)
- Inputs (campo de busca)
- Ícones (ações e navegação)

---

## 🔗 Navegação (Fluxo do Usuário)

O usuário poderá:

- Acessar a página inicial (Dashboard)
- Navegar para a seção de equipes
- Visualizar os dados de cada equipe em formato de cards
- Acessar a classificação do campeonato
- Utilizar o campo de busca para encontrar informações

---

## ⚙️ Considerações Técnicas

- A aplicação será desenvolvida com HTML, CSS e JavaScript
- O Bootstrap será utilizado como base para estilização
- A integração com API será feita futuramente na fase de implementação
