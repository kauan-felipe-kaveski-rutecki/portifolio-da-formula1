# F1 Teams Hub
Aplicação web para visualizar informações sobre equipes de Fórmula 1, incluindo história, títulos e pilotos. O sistema também utiliza uma API para exibir dados atualizados do campeonato.

## Objetivo
Centralizar informações sobre equipes de Fórmula 1 em um único lugar, facilitando o acesso a dados históricos e atuais.

## Funcionalidades
* Listar equipes
* Visualizar detalhes de cada equipe
* Exibir informações como história, títulos e pilotos
* Mostrar classificação atual da Fórmula 1 (via API)

## Tecnologias
* HTML
* CSS
* JavaScript
* Bootstrap 5.3
* Ergast API (Fórmula 1)

## Escolha do Framework CSS
O framework escolhido foi o Bootstrap 5.3.

A escolha foi baseada nos seguintes critérios:
* Responsividade: possui um sistema de grid eficiente que facilita a adaptação para diferentes dispositivos.
* Componentes prontos: oferece elementos como cards, navbar e botões, que agilizam o desenvolvimento.
* Documentação: bem organizada e fácil de utilizar.
* Atualização: versão moderna que não depende de jQuery.

O Bootstrap permite criar um layout limpo, organizado e responsivo, ideal para um sistema informativo.

## Escolha da API
A API escolhida foi a Ergast API, utilizada para obter dados reais da Fórmula 1.

Motivos da escolha:
* Compatibilidade com o projeto: fornece dados diretamente relacionados à Fórmula 1.
* Dados dinâmicos: permite exibir a classificação atual do campeonato.
* Facilidade de uso: não requer autenticação e possui endpoints simples.
* Gratuita: pode ser utilizada livremente.

A API será utilizada na página de classificação para exibir dados atualizados do campeonato.
